
Follow the steps below to get the project started:


Step 1: In the terminal, go to the project folder
$ cd FOLDER_NAME

Step 2: Enter frontend folder
$ cd frontend

Step2: Install node dependencies
$ npm install

Step 3: Install bower dependencies
$ bower install

Step 4: Exit out one directory
$ cd ..

Step 5: Enter middleware folder
$ cd middleware

Step 7: Install node dependencies
$ npm install

Step 7: Start the node server
$ node app.js
